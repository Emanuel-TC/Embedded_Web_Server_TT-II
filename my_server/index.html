Sobre el TT

Desarrollo de un prototipo de sistema embebido

Para el monitoreo de parámetros en microrredes eléctricas utilizando el concepto de IoT, con la finalidad de gestionar de una manera más eficiente la energía fotovoltaica proporcionada por un panel solar. Los parámetros a monitorear de la microrred son corriente y voltaje, mediante un sensor especializado, el cual es configurado mediante un sistema en chip (SoC) y enviados a un servidor implementado en un sistema embebido. El servidor permitie la consulta de los datos enviados de la microrred

Development of an embedded system prototype

For the monitoring of parameters in electrical microgrids using the IoT concept, in order to more efficiently manage the photovoltaic energy provided by a solar panel. The microgrid parameters to be monitored are current and voltage, through a specialized sensor, which is configured through a system on chip (SoC) and sent to a server implemented in an embedded system. The server allows the consultation of the data sent from the microgrid
